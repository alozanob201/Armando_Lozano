# Laboratorio 03 - Jerarquía de Memoria

Este repositorio contiene el desarrollo del Laboratorio 03 de Arquitectura de Sistemas, enfocado en el análisis de la jerarquía de memoria, latencia de acceso y comportamiento de caché.

## Contenido del repositorio

- `src/`: código fuente utilizado en el experimento de comportamiento de caché
- `benchmarks/`: comandos y resultados de las pruebas de latencia con lmbench
- `graficas/`: gráficas generadas a partir de los resultados
- `docs/`: documento final del laboratorio

## Entorno de trabajo

Para este laboratorio se utilizó una máquina virtual con Linux Ubuntu 24.04.2, arquitectura x86_64, 4 núcleos de CPU y aproximadamente 7.8 GiB de RAM.

## Experimentos realizados

1. Exploración de la arquitectura del sistema
2. Benchmark de latencia de memoria con lmbench
3. Visualización de la jerarquía de memoria
4. Comparación entre acceso secuencial y acceso aleatorio
